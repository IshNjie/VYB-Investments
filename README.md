# 📈 Stock Analysis

A lightweight Python module for fetching, visualizing, and benchmarking stock performance using **yfinance**, **pandas**, and **matplotlib**.

This repository contains a `StockAnalysis` class that simplifies:

- Retrieving historical stock data
- Plotting closing prices
- Comparing a stock against benchmark indices
- Normalizing returns for fair comparison

---

## 🧩 Features

- Fetch historical price data from Yahoo Finance
- Plot closing price trends
- Compare a stock against multiple benchmarks (User discretion)
- Normalized return comparison (all series start at 1)

---
